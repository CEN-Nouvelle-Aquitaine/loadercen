LoaderCEN 2.7

Améliorations:

1. Intégration des images drone:

  • Possibilité de charger les images drone du CEN-NA à partir des emprises de vols associées

2. Intégration des données cadastrales:

  • Possibilité de charger les parcelles cadastrales sur une commune en renseignant le code INSEE (dans la prochaine version l'interrogation directe par nom de commune sera possible)

3. Expérience utilisateur :

  • Application d'un style de transparence aux couches de dallages de manière à faciliter la visualisation des potentielles autres couches du projet

  • Application d'un zoom automatique sur l'étendue de la couche de dallage sélectionnée
